# Plan Instructions

Use this folder for implementation plans that turn roadmap outcomes into delivery-ready work.

## Skill

Use the [`implementation-planner` skill](../.codex/skills/implementation-planner/SKILL.md) for the planning workflow, including converting roadmap items into concrete deliverables and refining existing plans until they are implementation ready.

## Folder Rules

- Keep one plan per roadmap item or implementation slice unless there is a clear reason to split it.
- Treat documents in this folder as delivery artifacts, not high-level roadmap notes.
- Keep the detailed workflow, quality bar, and expected plan structure in the skill rather than duplicating it here.
