# Plan Template

Use this structure when drafting or refining a plan in `Plans/` unless the user asks for a different format.

1. Overview
2. Roadmap Source or Existing Plan Context
3. Scope
4. Non-Goals
5. Behaviour Scenarios
6. Deliverables
7. Technology Requirements and Decisions
8. Dependencies and Sequencing
9. Risks and Mitigations
10. Unknowns and Required Clarifications
11. Completion Checklist
