Review direction

Direction C1

Turn the homepage into a public bulletin that feels active and unmistakable.

This option keeps Direction C's campaign energy, but translates it into a more deliberate civic-broadcast header and a clearer rhythm between statement, evidence, and next step.

Why it works

The page reads like a designed announcement rather than an app shell.

Masthead clarity

The header behaves like a visible site frame, with distinct brand, navigation, and action areas.

Fast scanning

Condensed display type and bright markers keep deadlines, updates, and key reassurance easy to find.

Civic confidence

The palette feels more public-service modern than lifestyle, while staying energetic and current.

Implementation note

This direction would scale well if later public pages need a strong information-first header.

Prototype review only. Static HTML and CSS, not production Angular UI.
