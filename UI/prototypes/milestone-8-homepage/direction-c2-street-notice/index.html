Direction C2

Push the homepage toward a neighbourhood poster language with stronger public-facing energy.

This option makes Direction C feel more like a visible local campaign: bold headlines, stacked message bands, and a header that reads as an intentional public front door.

What shifts

The hierarchy becomes more poster-like while still explaining the service clearly.

Header presence

The masthead is layered and framed, so brand, navigation, and action feel designed together.

Campaign tone

Rust and sand create urgency without tipping into glossy marketing or startup styling.

Public readability

Big poster typography keeps calls to action and reassurance copy legible from the first glance.

Implementation note

This direction has the strongest "campaign leaflet" personality of the refinements.
